Template for PHP adder methods used when "Generate Adders" action is invoked.
Predefined variables will take the following values:
${IS_DOCTRINE_COLLECTION}   If field is annotated with `Doctrine\Common\Collections\Collection` takes the value of "doctrine" or empty string otherwise.
${FIELD_NAME}   The original field name as is.
${PARAM_NAME}   Singular form of field name(assuming that field name is plural). For example, `items` => `item`, `users` => `user`
${TYPE_HINT}   Type hint of param used in doc-comment, based on field name type hint. For example, if field is typehinted with `App\User[]`, then ${TYPE_HINT} is `App\User`, if field is of type `array` - `mixed`. FQCN used.
${TYPE_HINTED_PARAM}   If field is array of some class instances, then this `ClassName` used to type hint. Otherwise it is empty. Already includes param name. Not FQCN.
${METHOD_NAME}   Name of method being generated, based on field name. For example, if field name is `user` - method name is `addUser`(already prefixed).
${THIS_CLASS_NAME}   Name of context class. Not FQCN. E.g. `User` - if current class FQCN is `App\Entity\User`
